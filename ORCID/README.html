Description

This script fetches all of the "works" (papers, books, etc.) from the public profile of a given ORCID. It iterates over alist of ORCID IDs in a text file. It returns in a JSON file that is one big JSON array with each ORCID works record being an object in the array.
For errors or troubleshooting, contact John O'Connor at Boston College Libraries.

Requirements:

  1. Python 3.x.x
    • Mac/OSX users already have this installed on their computers. Skip Step 1 below.
    • Windows users can install it using the instructions below
  2. Requests library for Python
    • All users must install this library using the instructions below

To install requirements:

  1. Download and install the latest version of Python (3.6.4 as of 2018-03-15)
    1. Download file: https://www.python.org/ftp/python/3.6.4/python-3.6.4-amd64.exe
    2. Double click .exe file
    3. Check the "Add Python 3.6 to PATH" option at the bottom of the window
    4. Click "Install now"
    5. Wait for it to finish installing and close window
  2. Open a PowerShell window by searching for "powershell" in the windows search bar or pressing Alt+Shift, right clicking, and choosing "Open Powershell window here"
  3. Type "pip install requests" (without the quotes)
  4. Wait for it to finish installing and close window

To use this script:

  1. Ensure that all required software (above) is installed
  2. Download FacultyWorks.py and orcids.txt into a new folder
  3. Open orcids.txt in a text editor and paste each ORCID ID into that file
    • One ORCID per line
    • Delete the instructions line at the top of the file
  4. Open the folder that contains FacultyWorks.py and orcids.txt
  5. Open a PowerShell window by pressing Alt+Shift, right clicking in the folder, and choosing "Open Powershell window here"
  6. Type "python FacultyWorks.py" (without the quotes)
  7. Wait for the success message at the end to print
  8. Check that the folder now contains a new file called worksOutput.json
  9. Close Powershell window
